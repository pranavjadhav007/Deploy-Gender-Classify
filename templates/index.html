About Algorithm:

Here we use Artificial Neural Network to train the model. Processing tuner use for model extension. Keras Tuner suggest that alone Forehead Height and Forehead width ignites 95.8% neurons. So dosent require other parameters but still to increase accuracy, more parameters are use. A general classification is done by combining simple dense model and tuner model. Total 10,817 trainable parameters are present with Relu as an activation function for hidden layers. For prediction purpose we use Sigmoid as activation function. We use Adam optimizer and Binary Crossentropy as a loss function. With the help of very customized model we are able to achieve high accuracy by considering just two imp parameters. Our AI model works with 99.3% accuracy which helps in classifying whethere a person is Male or Female from its face measures.
